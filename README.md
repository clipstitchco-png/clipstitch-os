# ClipStitch Team Portal

A premium, static onboarding experience for ClipStitch designed to help new team members understand the company quickly and confidently.

## Structure
- index.html — home page
- pages/ — supporting onboarding pages
- style.css — shared design system and responsive layout
- script.js — interactions, theme toggle, search, animations, and org chart behavior
- assets/ — visual assets and icons
- downloads/ — placeholder download files

## Run locally
Open index.html in a browser, or serve the folder with a simple static server.

Example:
```bash
python -m http.server 8000
```

Then visit http://localhost:8000/
